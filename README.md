# 🛒 نظام إدارة المبيعات والمخزون (Sales & Inventory Management System)

نظام متكامل لإدارة المنتجات، تتبع المبيعات، وتحليل الأداء باستخدام واجهة مستخدم عصرية وسريعة.

## 🚀 المميزات
- **إدارة المنتجات:** إضافة، تعديل، وحذف المنتجات مع تتبع الكميات.
- **تتبع المبيعات:** تسجيل المبيعات اليومية وربطها بالمنتجات.
- **لوحة تحكم ذكية:** رسوم بيانية توضح أداء المبيعات وتوزيعها الجغرافي.
- **دعم اللغات:** واجهة كاملة باللغتين العربية والإنجليزية.
- **وضع الذاكرة:** يعمل النظام حالياً بوضع التخزين في الذاكرة (In-memory) لسرعة الأداء وسهولة العرض.

## 🛠️ التقنيات المستخدمة
- **Frontend:** React, Tailwind CSS, Lucide Icons, Framer Motion.
- **Backend:** Express.js (Node.js).
- **Charts:** Recharts.

## 📦 التشغيل المحلي
1. قم بتثبيت التبعيات: `npm install`
2. ابدأ خادم التطوير: `npm run dev`
